# Black-White-Waves-3JS

A mesmerizing WebGL animation featuring dynamic black and white waves created using Three.js and custom GLSL shaders. This project showcases fluid, organic wave patterns that evolve over time, perfect for creative coding challenges or as a visual backdrop.

## Features

- **Real-time Animation**: Smooth, continuous wave animations powered by GLSL fragment shaders.
- **Responsive Design**: Automatically adjusts to window size for optimal viewing on any device.
- **Black and White Aesthetics**: Striking monochromatic visuals with intricate wave distortions.
- **Lightweight**: Simple setup with minimal dependencies (only Three.js).

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Black-White-Waves-3JS.git
   ```

2. Open `index.html` in your web browser.

That's it! The animation will start automatically.

## Technologies Used

- **Three.js**: JavaScript 3D library for WebGL rendering.
- **GLSL Shaders**: Custom vertex and fragment shaders for wave generation.
- **HTML/CSS/JavaScript**: Standard web technologies for structure and interaction.
